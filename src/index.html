How to use this Site

  • The map can be repositioned by clicking on the map and dragging.
  • Zoom in and out using your fingers or alternatively you can use the mouse wheel.
  • Use the drawing tools to plot markers and add text. You can add up to four markers to the map.
  • Use the ruler tools to measure.

About

this side project is just a sample of the project I mainly work at Buchanan Computing, Traffweb a web mapping application used by our clients to dipslay the restrictions or collisions in their boundary.
